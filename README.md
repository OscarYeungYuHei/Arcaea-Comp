# Arcaea Battle Song Simulator

A live-stream-ready Arcaea battle song pick/ban simulator. 1920×1080 layout with in-game jacket art.

## Usage

Open `arcaea-battle-simulator.html` in a browser.

## Structure

```
├── arcaea-battle-simulator.html  # main app
├── arcaea_illustrations/         # song jacket images
├── package.json
├── package-lock.json
└── .gitignore
```
